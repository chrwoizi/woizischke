5.3 Modellauswahl

Beim Modelchooser kann sich der Spieler das Model aussuchen mit dem er das Spiel spielen möchte. Hierbei rotiert das Model um die Z-Achse.

Die Balken Speed, Shield, Health und Fuel geben Werte für die Eigenschaften des Models im Spiel an. So gibt z.B. Fuel an, wie viel Fuel das Model für den Boost beim Start des Spiels hat.

Mit Hilfe der Maus kann man das Model so rotieren wie man es gerne hätte. Das heißt, man kann es von allen Seiten betrachten. Nach der Auswahl rotiert das Model dann in seiner aktuellen Lage weiter um die Z-Achse.

(a) Chooser normal (b) Chooser Rotation
